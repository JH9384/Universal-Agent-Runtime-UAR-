# UOR Compatibility

UAR supports interoperability with UOR-style systems.

## Capabilities (planned)

- Export objects to UOR-compatible schemas
- Import UOR-style objects into UAR graphs
- Preserve identity mappings where applicable

## Non-Goals

- UAR does not implement full UOR semantics
- UAR does not require UOR compliance
- UAR operates independently as an agent runtime

## Philosophy

UOR is:

- a useful abstraction
- an interoperability layer
- not a system constraint
- not the system identity
