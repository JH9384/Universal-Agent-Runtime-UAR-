# UAR Architecture

## Core Layers

### 1. Ingestion Layer

- markdown
- text
- file-based inputs

### 2. Object Normalization Layer

- numbers
- text
- document sections

### 3. Graph Layer

- nodes
- edges
- composition

### 4. Runtime Layer

- registered runtime functions
- sandboxed execution

### 5. Agent Layer

- execution orchestration
- trace
- verification hooks

### 6. Compatibility Layer

- UOR bridge
- import/export adapters

---

## Key Principle

UAR is:

- runtime-first
- agent-driven
- graph-executed

UAR is not:

- an object-first system
- a strict UOR implementation

Objects exist to support execution, not to define the system identity.
