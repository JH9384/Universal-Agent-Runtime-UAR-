# UAR Operator Dashboard

## Purpose

The operator dashboard exposes live runtime continuity cognition and operational orchestration.

## Planned Views

- Continuity Fabric Overview
- Replay Timeline
- Topology Cognition
- Restoration Orchestration
- Governance and Escalation
- Fabric Forecasting

## Runtime Streams

The dashboard is expected to consume:

- fabric_stream
- replay_stream
- topology_stream
- health_stream

## Rendering Goals

- Live topology rendering
- Replay restoration visibility
- Anomaly propagation overlays
- Continuity-health visualization
- Runtime-fabric governance controls
