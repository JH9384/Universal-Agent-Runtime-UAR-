# Continuity Equilibrium Metrics Schema

## Purpose

Defines the canonical metrics used during Ω-V stabilization and convergence certification.

## Broadcast Metrics

| Metric | Purpose |
|---|---|
| event_rate | operational propagation volume |
| queue_pressure | propagation backlog pressure |
| subscriber_lag | delayed subscriber detection |
| broadcast_drop_rate | propagation suppression pressure |

## Topology Metrics

| Metric | Purpose |
|---|---|
| topology_stability | graph equilibrium measurement |
| mutation_velocity | mutation pressure tracking |
| graph_pressure | topology propagation pressure |
| equilibrium_score | stabilization convergence |

## Governance Metrics

| Metric | Purpose |
|---|---|
| governance_pressure | escalation pressure |
| escalation_rate | governance amplification rate |
| policy_entropy | governance instability |
| response_latency | stabilization response time |

## Replay Metrics

| Metric | Purpose |
|---|---|

| reconciliation_velocity | reconciliation effectiveness |

## Restoration Metrics

| Metric | Purpose |
