# Runtime Quickstart

## Setup

```bash
bash scripts/runtime_quickstart.sh
```

## Runtime API

After startup:

- Health: http://127.0.0.1:8080/health
- Telemetry: http://127.0.0.1:8080/telemetry
- WebSocket: ws://127.0.0.1:8080/ws/runtime

## Smoke Validation

```bash
bash scripts/runtime_smoke.sh
```

## Strategic Direction

This quickstart turns UAR into a locally runnable operational runtime surface.
